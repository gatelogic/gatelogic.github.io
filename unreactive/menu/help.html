Commands for UnReactive

General Commands

getuuid <MC Username>
Fetches your UUID from the user database.

Moderation Commands

ban <@username> <reason>
Bans a user from the server.
kick <@username> <reason>
Kicks a user from the server.
clear <amount>
Clears a certain amount of messages from the channel.
cmute (unlock)
Locks/Unlocks a channels @everyone perms.
nickname <@username> <nickname>
Gives a user a new nickname.

Misc Commands

embed
Generates an embed. Use ; to create new lines.
giverole <@username> <@role>
Gives a role to a user.
invite
Puts an invite link in chat.
userinfo (@username)
Gives info on you/another user.
